Описание механик и скриптов в программе:
1. DragItem класс состоит из ивентов OnMouseDown (Up, Drag) и метод GetMouseTransform. Скрипт отвечает за перетаскивание предмета
2. ItemHitTheTrigger класс состоит из двух методов (TriggerEnter, TriggerExit), во время попадания предмета в область тригера, он запускает анимации через DOTween на уменьшение объекта до задоного scale
3. ScrollCheck класс отслеживающий, можно ли проводить скролинг сцены
4. CameraBounds класс который скролит сцену и контролирует, чтобы она не выходила за границы

Игровой процесс 
1. Поднятие предмета выполняется через нажатие и зажатия предмета на сцене
2. Чтобы скролить сцену с предметом, нужно просто вести предмет в нужную сторону по X, в которую вы хотите скролить
3. Без предмета, можно зажать правую или левую часть экрана, чтобы скролить в соответствующую сторону
4. Для активации класса, который будет уменьшать айтем, нужна перетащить предмет над одной из тумбочек и отпустить. Тумбочки находятся с левого края сцены и с правого.
